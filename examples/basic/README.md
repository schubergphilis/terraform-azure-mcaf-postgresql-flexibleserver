# Basic Example

This example demonstrates a minimal configuration for the PostgreSQL Flexible Server module.

## Features

- Single PostgreSQL Flexible Server with default settings
- Private endpoint connectivity
- Customer-managed key encryption
- High availability enabled (default)

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Requirements

- Azure subscription
- Terraform >= 1.0
- AzureRM provider >= 3.0
